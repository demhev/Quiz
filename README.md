# 🎮 Genshin Impact Quiz – Web-приложение для викторин

**Интерактивный опросник по вселенной Genshin Impact и не только.**  
Проект написан на **Python + Flask** с использованием **SQLite** и стилизован под тематику игры.  
Позволяет проходить готовые викторины, а также **создавать новые опросы прямо через веб-интерфейс**.

---

## 📌 Возможности

- 📋 **Просмотр всех доступных викторин** на главной странице.
- ❓ **Прохождение викторины** – выбор ответов, подсчёт правильных результатов.
- 🆕 **Создание нового опроса** – введите название, и он появится в списке (можно дальше наполнять вопросами).
- 🎨 **Стилизация** в тёмных «геншиновских» тонах с золотыми акцентами.
- 📱 **Адаптивный дизайн** на Bootstrap 5.

## 🚀 Быстрый старт

### Клонирование репозитория
```bash
git clone https://github.com/ваш-username/quiz-genshin.git
cd quiz-genshin
```

### Установка зависимостей (рекомендуется виртуальное окружение)
```bash
pip install flask
```

### Инициализация базы данных
```bash
python db_scripts.py
```
Скрипт создаст файл `quiz.sqlite` и заполнит его двумя викторинами:
- **Random opros** (общеобразовательные вопросы)
- **О знании версии 6.7 Genshin Impact** (каноничные вопросы по игре)

### Запуск веб-сервера
```bash
python stranichli.py
```
Откройте в браузере `http://127.0.0.1:5000`

## 🔧 Как это работает

### База данных
Три таблицы:
- `quiz` – названия опросов
- `question` – вопросы с правильным и тремя неправильными ответами
- `quiz_content` – связь опросов и вопросов (многие ко многим)

### Создание нового опроса (в разработке)
На главной странице есть форма, которая отправляет POST-запрос на `/new_quiz`.  
В обработчике вызывается функция `add_quiz(name)` из `db_scripts.py`, которая вставляет новую запись в таблицу `quiz`.  
После редиректа на главную новый опрос отображается в списке.
