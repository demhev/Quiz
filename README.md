# 🎮 Genshin Impact Quiz Web App

**Интерактивный опросник** на Python + Flask с возможностью создавать собственные викторины прямо на сайте.  
Стилизован в духе игры Genshin Impact, но подходит для любых тем.

---

## 📌 Возможности

- 📋 **Главная страница** – список всех доступных викторин.
- ❓ **Прохождение опроса** – выбор ответов, автоматический подсчёт правильных.
- 🆕 **Создание новых опросов** – через веб-форму можно добавить название и любое количество вопросов (каждый вопрос содержит правильный и 3 неправильных варианта).
- 🎨 **Тёмный дизайн** с золотыми акцентами, адаптивный.
- 🗃 **SQLite** – все данные хранятся локально в файле `quiz.sqlite`.

---

## 🛠 Технологии

- **Backend:** Python 3.8+, Flask
- **База данных:** SQLite (sqlite3)
- **Frontend:** Bootstrap 5, кастомный CSS, JavaScript (динамическое добавление вопросов)
- **Шаблонизация:** Jinja2

## ⚙️ Установка и запуск

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/ваш-username/quiz-genshin.git
cd quiz-genshin
```

### 2. Создайте и активируйте виртуальное окружение

#### Windows (CMD)
```bash
python -m venv venv
venv\Scripts\activate
```

#### Windows (PowerShell)
```bash
python -m venv venv
venv\Scripts\Activate.ps1
```

#### macOS / Linux
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установите зависимости
```bash
pip install flask
```

### 4. Инициализируйте базу данных
Скрипт создаст файл `quiz.sqlite` и добавит две тестовые викторины.
```bash
python db_scripts.py
```

### 5. Запустите веб-сервер
```bash
python stranichli.py
```

Откройте браузер и перейдите по адресу: [http://127.0.0.1:5000](http://127.0.0.1:5000)

## 🧠 Как это работает

### База данных
- **quiz** – хранит названия опросов (`id`, `name`).
- **question** – хранит вопросы и варианты ответов (`id`, `question`, `answer`, `wrong1`, `wrong2`, `wrong3`).
- **quiz_content** – связывает опросы и вопросы (`id`, `quiz_id`, `question_id`).

### Создание опроса
- Форма на странице `/create_quiz` принимает название и произвольное количество вопросов.
- При отправке сервер вставляет запись в `quiz`, затем каждый вопрос в `question`, и создаёт связи в `quiz_content`.
- После сохранения пользователь перенаправляется на главную страницу, где новый опрос уже отображается.

### Прохождение опроса
- При GET-запросе на `/quiz/<id>` из базы извлекаются все вопросы, принадлежащие этому опросу, и отображаются с радиокнопками.
- После отправки формы (POST на `/quiz/<id>/check`) сравниваются ответы пользователя с правильными и выводится результат.

## 🐛 Частые проблемы

### Ошибка `ModuleNotFoundError: No module named 'flask'`
- Убедитесь, что виртуальное окружение активировано.
- Проверьте установку Flask: `pip list | grep flask`.
- Если не установлен, выполните `pip install flask`.

### База данных не создаётся
- Запустите `python db_scripts.py` вручную.
- Проверьте права на запись в текущей директории.

### Не отображаются стили
- Убедитесь, что файл `static/style.css` существует и подключён в `base.html`.
- Очистите кэш браузера (Ctrl+F5).
